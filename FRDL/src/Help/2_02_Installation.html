Installing FRDL

FRDL is a Java application which means it can work on any operating system which has Java installed on it. You probably have Java already installed; FRDL requires you to have at least Java version 1.5 but ideally you should have version 1.6. If you do not have Java or you need an update, it is available from
java.sun.com/javase/downloads/index.jsp.
You will need the Java SE Runtime Environment (JRE).

Installing FRDL on a PC

FRDL has been tested on Windows 2000, XP and Vista.

Windows Download the file setup FRDL xxx.exe to your computer from www.flymicro.com/frdl and double click on it to run the installer.

Other operating systems (and Windows, if the above doesn't work). Download the file setup FRDL xxx.jar to your computer from www.flymicro.com/frdl and double click on it to run the installer. (A .jar file is actually just a zip file recognizable to Java, but the .jar installer may not work correctly on Vista...)

In both cases the xxx indicates the version. Versions of FRDL up to and including 0.24 should be uninstalled before installing a newer version. For versions 0.25 and greater it is perfectly OK to install over an earlier version of FRDL. In either case no configuration files, logger files or output igc files will be affected.

On Vista you may still get a message from the "Program compatibility assistant": "This program might not have installed correctly" with the Details: IzPack SFX, publisher: Igor Pavlov, Julien Ponge. This is OK, they are the authors of the installer and probably FRDL has actually installed OK.

The installer does nothing 'clever' except to put the required files in a place of your choice and creates shortcuts so you can easily start FRDL.

Uninstalling FRDL on a PC

Select Uninstall FRDL from the start menu.

If, for some reason this does not work, search your system for FRDL.jar and delete the folder it is in along with all it's sub-folders, and any shortcuts pointing at these folders (eg in your START menu). FRDL is now completely removed from your system.

Uninstalling FRDL will not affect any of the championship files, logger backups or .igc files it may have made while in use.

Installing and uninstalling FRDL on Mac, Linux, Solaris Etc.
