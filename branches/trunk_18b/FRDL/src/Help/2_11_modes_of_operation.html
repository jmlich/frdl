Modes of operation

Full mode

FRDL is normally in full mode, which means you have full control over all settings.

If you set a password in Championship settings then FRDL will be in full mode with any loggers it encounters which have a matching password.

A conpetition organization will normally be running in in full mode with a password. In this mode, FRDL copies all its key configuration data like task window settings to the logger.

Download-only mode

If the passwoed doesn't match, you are asked if you want to go into Download-only mode.

This mode is designed for pilots and team leaders at championships so they can download the exact same data as the organization.

The organizer's settings are retrieved by FRDL in download mode and used to produce the resulting .igc file output on the pilot or team leader's computer using the exact same settings as the organization.

Getting out of Download-only mode

The password should be entered in FRDL. It would be normal for an organizer to release the password at the end of a championship.
