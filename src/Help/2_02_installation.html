Installing FRDL

FRDL requires Java version 6 to be installed on your computer, and it probably is already. If you do not have this, it is available from http://java.sun.com/javase/downloads/index.jsp. You need the Java SE Runtime Environment (JRE).

Installing FRDL on a PC

FRDL has been tested on Windows 2000, XP and Vista.

Download the file FRDLxx.zip file to your computer. (the xx indicates the version).

Unzip the contents of the file to a location of your choice on your computer. The folder thus created should contain the file FRDLxx.jar and the sub-folder lib containing various libraries FRDL needs to run.

Double click on FRDLxx.jar to start FRDL. It is recommended a shortcut is created and placed in a convenient place such as the desktop.

Uninstalling FRDL on a PC

Delete the folder containing the file FRDLxx.jar and the sub-folder lib.
Delete any shortcuts you may have made to FRDLxx.jar
FRDL is now completely removed from the PC.

Installing FRDL on Mac, Linux, Solaris Etc.

In principle, FRDL should work on all these platforms, however it is currently not tested on any of them. It is known that FRDL will not work on a Mac with OSX 10.4.11 because there is no Java version 6 available for it (only version 5), however, version 6 is available for more modern Mac OS's.

The developer would be interested to hear of any experiences of using FRDL on non-Windows platforms.
